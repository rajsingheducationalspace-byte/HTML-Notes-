# What is Tag? Ans. Tag is a special keyword which tell the browser how to display the things in the website page.

# What is elements? Ans. elements are the mixture of tags and contents.

# What is attributes? Ans. attributes are the extra piece of informations which are added in the html tag. 

# What is semantic tag? Ans. Semantic tags are those tags which has clear meaning.

**Heading tag, paragraph tag, br tag, pre tag, a href tag, img tag, ordered and unordered tag, table tag, form tag,**

 so first of all we have to make a boiler plate for it which consist many things. let's take a example. 

 **<!DOCTYPE html>** - it tells the browser that what the documentation type of the language.
**<html lang="en">** - it means language we use like english, we can also change it into hindi.
**<head>** - this is the head where meta data with title are written.
  **<meta charset="UTF-8">** this line means character set which means it tell the browser which kind of character are enable by the webpage. like utf-16 is use for special character and symbols.1
  **<meta name="viewport" content="width=device-width, initial-scale=1.0">**
  **<title>Your Website Title</title>** this means the title of the webpage like name of the webpage.
**</head>**
**<body>** this is body tag which means the part of the webpage which is visible to us. where all the images and information of website is display

  **<h1>Welcome to My Website!</h1>** This is one the tag which we mention above. it is generally a heading tag. there are 6 heading tag from h1 to h6 where h1 means most important heading to the browser and h6 means least important heading to the browser. by using right heading tag a website can easily rank in the browsers like google.
  **<p>This is my first web page.</p>** This is paragraph tag where we write the paragraph of the website or blog page. 
  **<br>** this is br tag which help to break the line of the paragraph, table, form etc.
  **<pre>this is pre</pre>** this is pre tag which help to display the same content written during the web making. it allow to display the change like space, next line, to the webpage.
  **<b> this is bold</b>** this is bold tag which help to bold the text. 
  **<i> this is italic</i>** this is italic tag which help to show the text in cursive manner.
  **<strong> this is strong</strong>** this is strong tag and help to show the browser about the importance of the word.
  **<mark> this is mark tag</mark>** this is mark tag which help to highlight the text.
  **<small> this is small tag</small>** this is small tag which help to show the text in small with comparizing with other text.
  **<del> this is delete tag</del>** this tag is generally use in the advertisement banner to show the discount amount.
  **<sup> this is sup </sup>** this tag is use during writing the scientific formulas which allow to write the word as power of it.
  **<sub> this is sub</sub>** this tag is also use during writing the scientific formulas which allow to write the word as base power of it.
  **<a href=""> this is a tag</a>** this tag is use to mention the link of site or other thing to it
  **<link rel="stylesheet" href="" this is link tag>** this tag is use to link the two or more pages together.
  **<img src="" alt="" this is image tag>** this is image tag which help to display the image of the website with the server. there is alt in the image element which help to display the text when the image is not load.
  **<table> this is table</table>** this is table tag which help to make a table of something. there are many sub tags which are written the table tag.
  **<ol> this is ordered tag</ol>** this tag is known as ordered list where we write the content related to the order.
  **<ul> this in unordered tag </ul>** this is unordered tag which where does not have ordered of the tag like it shows in bullet
  **<tr> this is table row</tr>** this is table row tag which help to make different rows of table. 
  **<th> this is table heading</th>** this is table heading which help to mention the heading of the table 
  **<td> this is table data</td>** this is table data where the data of the table is put.

  **<form action="" class=""></form><form action="" class=""> this is form tag</form>** this tag is use to make a form of a particular thing like application form etc. 
  **<button> this is button</button>** this is button tag and help to make a button in the form. 
  **<iframe src="" frameborder="0"> this i frame tag</iframe>** this i frame tag is use to add the third party content like video, audio to the website with embeded code . 
 
  we can also do some inline styling in the middle of line. we can use **<span> this tag for doing styling in the line.</span>**
